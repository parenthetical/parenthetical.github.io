Journal entry for March 8, 2024

My work hasn’t been very focused lately. I’m having fun with friends and climbing though, it’s been re-energizing me. My parents are visiting me in Lisbon so I’ll be spending a few days with them and trying to fit in work during off hours.

I’m currently reading The Hunt for the Missing Data Type from Hillel Wayne. The confusion of graph libraries is definitely something I’ve run into. I wonder if it would be possible for a programming environment to provide a highly generic interface which can be specialized using annotations to a specific implementation that’s going to be performant for the use case.
